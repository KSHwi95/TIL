> 두 테이블에서 공통된 이름의 칼럼이 `키 변수`가 된다.

| **이름** | **나이** |
| :------: | :------: |
|  김준수  |    16    |
|  최수빈  |    18    |
|  최사랑  |    25    |

| **이름** | **성별** |
| :------: | :------: |
|  김준수  |    남    |
